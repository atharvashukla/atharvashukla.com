How is this site deployed?

  • I host using Hostinger
  • I push the changes to my personal website repo on Github
  • I use Webhooks to connect Hostinger and Github

I simply write in a markdown file and the deploy script converts that to html and links it onto the front page. The title of the post becomes the title of the link and the filename becomes the path.

I use Alfred’s Hotkey workflow to add a keybinding (⌃⌥⌘D) to run the Makefile that compiles and deploys.
