- add oil to pan
- add jira + patti + onion to pan
- grind adrak and add
- grind lehsun and add
- (add adrak, lehsun paste after the onion is brown)
- then add kaju
- add tomatoes
- add salt, 1-2 spoons (big spoon not table spoon)
- add dhaniya, 2 tbsp
- add mirchi, 1.5 tbsp
- add haldi, 1-2 tbsp
- add water, 1 glass, keep mixing
- about 10-15 mins
- let it cool (in water)
- then add to mixer
- add some oil to kadhai
- add kashmiri laal, with some water mixed in
- add tej patta
- add masala and cover it. this will take some time.
